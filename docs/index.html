my-stt-tts

A local voice assistant for macOS Apple Silicon — wake word → on-device speech-to-text → an LLM → natural text-to-speech, in German, French & English. View on GitHub →

Hear the voices

Try it

uv sync --extra all && ./mstt --type — no API key needed (uses the Claude CLI). See the README.

Voices are Piper models (CC-licensed); the project is Apache-2.0. Built on parakeet-mlx (STT), Piper / Kokoro / macOS say (TTS), and Claude.
